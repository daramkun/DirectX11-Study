# Create Render-Target-View
DXGI 교환사슬에는 화면에 출력되고 있는 프론트버퍼와 메모리에서 렌더링이 수행되고 있는 백버퍼가 있습니다. 프론트버퍼에 백버퍼의 내용을 옮기려면 교환사슬의 ```Present``` 함수를 호출하면 되지만 DXGI의 백버퍼 객체에 Direct3D 11 API를 이용해 렌더링을 하려면 어떻게 해야 할까요?

DXGI 표면 객체와 Direct3D 11 텍스처 객체는 서로 호환됩니다. 따라서 DXGI 교환사슬로부터 백버퍼 표면 객체를 Direct3D 11의 2D 텍스처 객체 형식으로 가져온 뒤, 이를 이용해 렌더타겟 뷰를 생성해주면 여기에 렌더링을 수행할 수 있게 됩니다.

일단 이 챕터에서는 렌더링을 수행하지 않고 백버퍼를 초기화하는 과정만 해봅니다.
